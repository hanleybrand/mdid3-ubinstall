mdid3-ubinstall by hanleybrand

Setting up mdid3 on an ubuntu vm, or whatever

This is a simple project, to make shell scripts to simplify the installation of MDID3 on a Lucid Lynx VM for the purpose of doing development, testing MDID3, etc.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/hanleybrand/mdid3-ubinstall

get the source code on GitHub : hanleybrand/mdid3-ubinstall
